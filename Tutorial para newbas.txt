Tutorial para newbas:

Antes de tudo, verifique com os cologuinhas se eles mandaram algum commit, ou, de o comando |git status| e ele vai te mostrar se vc está algum commit atrasado.
Caso isso aconteça, dê um git pull e siga as instruçoes do powershell para resolver os erros(se der algum)

Para mandar suas modificações pro git, dê um |git add .|, seguido de um | git commit -m "Descrição do q vc mexeu" |
Depois disso, dê um | git push | , ou | git push origin main | (caso de erro de não reconhecer sua branch)

PS: O comando | git status | é mt forte pra resolver erros e saber o q está acontecendo com o git

PS2: A extensão GitLens -- Git supercharged te mostra quem deu qual commit, te da o link do commit para o github e da mais detalhes, caso precise pedir para alguem o que a pessoa fez
